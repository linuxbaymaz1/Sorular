
Bu proje, Go (Golang) ve SQLite kullanarak bellek içi (in-memory) veritabanı işlemlerinin (Insert, Update, Delete) teorik maksimum sınırlarını test eden bir performans aracıdır.



## ⚙️ereksinimler

- **Linux:** `sudo apt-get install build-essential`
- **Windows:** MinGW veya TDM-GCC kurulu ve PATH'e eklenmiş olmalıdır.
- **macOS:** `xcode-select --install`


## Kurulum

   git clone [https://github.com/KULLANICI_ADIN/sqlite-benchmark.git](https://github.com/KULLANICI_ADIN/sqlite-benchmark.git)
   cd sqlite-benchmark
   go mod download
   go run main.go
